Les objets

La notion d'objet avec Katalon

Katalon utilise les objets pour interagir avec la page Web. Ils correspondent aux éléments web capturés pendant l'enregistrement : champs texte, bouton, menus déroulant...

Zoom

The Object Repository

Après l’enregistrement, on retrouve ces objets stockés dans l'Object Repository

Zoom

L' Object Repository permet de gérer en un même endroit tous les identificateurs d'éléments de l'interface.

Un atout indispensable en cas de refonte de l'ergonomie. Au lieu de modifier chacun des tests utilisant un même objet, on ne modifie que la définition de l'objet ayant subi une modification. En programmation, on parle de factorisation.

Capturer les objets en mode Spy Web

Pour alimenter l'Object Repository en dehors d'un enregistrement, il faut utiliser l'outil de capture : « Spy Web ».

L'utilisateur sélectionne alors, en s'aidant d'une preview, les objets qui l'intéressent (nul besoin de mettre le nez dans le code).

Capturer les éléments visuels et cliquables : "Alt + ù" en mode Spy Web

A quoi ressemble un objet capturé ?

Zoom

Il est toujours possible (et parfois conseillé) de modifier le chemin ou renommer un objet :

  • soit directement pendant l'enregistrement ("Show Captured Objects")

  • soit en double cliquant sur l'objet à partir de l'Object Repository

Mais pour cela il est indispensable d'avoir certaines notions de XPath
