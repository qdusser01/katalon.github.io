Présentation de l'Automate

Automatiser les tests sur plusieurs types de SUT

Katalon est un automate qui permet d'interagir avec :

  • des IHM web (« traditionnelles » et riches)

  • des applications mobiles

  • des webservices

Zoom

Mode opératoire - Objectifs du cours

  1. Enregistrer des actions utilisateurs (outil de Capture / Playback)

  2. Générer un script

  3. Re-exécuter le script : Tester le rejeu des actions enregistrée

  4. Ajouter les vérifications : Elles constituent le test !

  5. Re-exécuter le script : Tester son script de test

  6. Améliorer ses scripts : Créer des variables, synchroniser

    • plus complets

    • plus pertinents

    • plus robustes

Un outil pour tous ?

L'outil a été conçu pour être utilisé aussi bien par des automaticiens que par des testeurs ne sachant pas programmer.

Les cas de test peuvent donc être conçus en mode « manuel » ou en mode « script » (à noter que les scripts sont en Groovy).

Utiliser cet outil peut être un bon moyen de s'initier en douceur aux joies du code.
