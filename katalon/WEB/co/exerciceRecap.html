Question

TC02 : Création d'un événement avec rappel

Pré-requis :

  • L'application WebCalendar est configurée pour que l'interface soit en français.

  • Les utilisateurs « Florence Warnet », « Lydia El Haik » et « Jean Roche » existent.

  • Le calendrier de l'utilisateur ne doit pas comporter d'événements susceptibles d'être en conflit avec l'événement à créer.

#

ACTION

RÉSULTAT ATTENDU

1

Se connecter à l'application WebCalendar.

URL = http://localhost/WebCalendar-1.2.5-auto/login.php

La page de login de WebCalendar est affichée.

2

Se loguer avec l'utilisateur « José Amaris » :

  • Dans le champ « Identifiant utilisateur », saisir la valeur « jamaris »

  • Dans le champ « Mot de passe », saisir la valeur « jamaris »

  • Cliquer sur le bouton « Connexion »

Le calendrier de l'utilisateur est affiché.

3

Sélectionner le menu « Événements », puis cliquer sur le sous-menu « Ajouter un événement ».

L'écran « Ajouter un événement » s'affiche. L'onglet « Détails » est présélectionné.

4

Saisir les détails de l’événement :

  • Dans le champ « Description résumée : », saisir la valeur : « Formation »

  • Dans le champ « Description détaillée : », saisir la valeur « Formation à la recette »

  • Dans le champ « Lieu : », saisir la valeur : « EQL »

  • A l'aide des listes déroulantes « Date : », sélectionner la date du jour

  • Sélectionner « événement permanent »

  • Dans la liste déroulante « Accès », sélectionner la valeur « Public »

  • Dans la liste déroulante « Priorité », sélectionner la valeur « 1-Haute »

  • Sélectionner la catégorie « Formation »

5

Cliquer sur l'onglet « Rappels ».

L'onglet « Rappels » s'affiche.

6

Saisir les caractéristiques du rappel :

  • Cliquer sur le bouton radio « Envoyer un rappel » – « Oui »

  • Cliquer sur le bouton radio « Délai »

  • Saisir un délai de 0 jour, 1 heure et 0 minute.

  • Cliquer sur le bouton radio « Avant »

  • Cliquer sur le bouton radio « Début »

9

Cliquer sur le bouton « sauvegarder ».

Le calendrier de l'utilisateur est affiché.

Dans la cellule correspondant à la date du jour, le lien « Formation » est affiché.

10

Cliquer sur le lien « Déconnexion »

La page de déconnexion s'affiche
