Utiliser les mots clés Gherkin

En utilisant les mots clés Gherkin anglais, complétez ce script.

Feature : Authentification

Scenario  : Vérifie la bonne connexion d'un utilisateur enregistré

Given je me trouve sur la page d'authentification du site

Given je dispose d'un compte  utilisateur « ginette »

When je saisis mon identifiant « ginette » dans le champ « Login »

When je saisie mon mot de passe « secret » dans le champ « Password »

When je clique sur le bouton « Connexion » du formulaire

Then je suis authentifié sur le site

Then je suis redirigé vers la page « Mon compte »
