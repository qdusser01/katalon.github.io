Les actions

Les actions enregistrées

Lors de l'enregistrement, les actions de la souris ou du navigateur sont prises en compte (clic droit, clic gauche, ouverture d'une fenêtre,...).

Néanmoins, certaines actions, plus complexes, doivent être ajoutées après l'enregistrement, comme le survol de la souris, la gestion d'une pop up,...

La liste d'actions

Parmi la liste des actions proposées par l'automate, on distingue :

  • Les actions du type comportement utilisateur

    • Click

    • Drag and drop

    • Mouse over

    • Navigate

    • ...

  • Les actions de synchronisation

    • Wait for element visible

    • Wait for alert

    • ...

  • Les actions de localisation d'éléments

    • Get text

    • Get URL

    • ...

  • Les actions du type assertions

    • Verify Element Present

    • ....

Ajouter une action manuellement

Pendant l'enregistrement :

  • Mettre en pause l'enregistrement

  • Cliquer sur Add

  • Choisir l'action souhaitée

Éditer le cas de test existant :

  • Cliquer droit sur le pas précédent l'action à ajouter

  • Cliquer sur Add --> WebUI Keyword action

  • Choisir l'action souhaitée

Actions supplémentaires

  • Mouse Over : Survole de la souris Exemple : sur un menu pour qu'il se déroule

  • Check / Un-Check : coche ou décoche une checkbox ou un bouton radio

  • Accept/Dismiss Alert : Gère un pop-up : accepte ou annule l'opération proposée par l'alerte

  • Switch to Window (Title / URL) : change le focus sur un autre onglet du navigateur
