Intérêts de la variabilisation des inputs

A quoi servent les variables ?

Les différents types de variables dans Katalon

Il existe trois types de variables :

  • privées

  • locales

  • globales

La déclaration dépendra de l'usage qui est fait de la variable

Les variables privées

Déclaration de la variable directement dans le script du Cas de Test

Les variables locales

Déclaration de la variable directement dans l'onglet "Variables"du Cas de Test

Les variables globales

Déclaration de la variable au niveau du profil de l'utilisateur

Les variables globales sont communes à tous les cas de tests d'un même profil, contrairement aux variables locales et privées qui ne sont accessibles qu'au cas de test où elles sont déclarées
